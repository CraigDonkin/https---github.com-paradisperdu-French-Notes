All appreciation verbs need a definitive article to introduce their direct object. 

So you would use les pates instead of des pates when saying I love pasta etc. 

## -er verbs

| je         | e    |
| ---------- | ---- |
| tu         | es   |
| il/elle/on | e    |
| nous       | ons  |
| vous       | ez   |
| ils/elles  | ent  |

## -ir verbs

| je         | is     |
| ---------- | ------ |
| tu         | is     |
| il/elle/on | it     |
| nous       | issons |
| vous       | issez  |
| ils/elles  | issent |
